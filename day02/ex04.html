table TAG

TABLE : 문서에 표를 그린다

  • table : 전체 표를 나타내는 태그, 내부에 다른 관련 태그를 포함한다
  • tr : table-row, 표 내부의 한 줄을 형성한다
  • td : table-data, 줄 내부에 데이터를 작성할 수 있는 한 칸을 만든다
  • th : table-headline, td와 동일하지만 중요항목이므로, 가운데 정렬과 굵게 속성 적용
  • thead : 테이블의 상단 영역
  • tbody : 테이블의 본문 영역
